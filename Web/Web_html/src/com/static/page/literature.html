迎寒祭月 :
“中秋”一词，最早见于《周礼》。《礼记·月令》记载：“仲秋之月养衰老，行糜粥饮食。” 每逢中秋夜，都要举行迎寒和祭月的仪式。 设大香案，摆上各式祭品，其中月饼和西瓜必不可少. 明代《帝京景物略》就曾记载：“农历八月十五日祭月，其祭果饼必圆，分瓜必牙错瓣刻之， 如莲华。”

观潮 :
在古代浙江一带，除赏月外，观潮可谓另一中秋盛事。每当农历八月十五日至十八日，潮势汹涌澎湃，比平时大潮更为奇特，潮头如万马奔腾、山飞云走，撼人心目，历代文人多有题咏。 观潮早在汉代枚乘的大赋《七发》中便有相关记述：“将以八月之望，与诸侯远方交游兄弟，并往观涛乎广陵之曲江。至则未见涛之形也，徒观水力之所到，则恤然足以骇矣。观其所驾轶者，所擢拔者，所扬汩者，所温汾者，所涤汔者，虽有心略辞给，固未能缕形其所由然也。” 宋神宗熙宁六年（1073年）中秋，时任杭州通判的苏轼作五首七言绝句《八月十五日看潮》，亦为观潮名作。

燃灯 :
中秋之夜，天清如水，月明如镜，可谓良辰美景，美不胜收。然而对此人们并未满足，便有“燃灯”以助月色的风俗。在湖广一带有用瓦片叠于塔上燃灯的节俗，在江南一带则有制灯船的节俗。 中秋夜，灯内燃烛，用绳将灯系于竹竿上，或用小灯砌成字形，挂于家屋高处，俗称“树中秋”或“竖中秋”。富贵之家所悬之灯，高可数丈，家人聚于灯下欢饮为乐；平常百姓则竖一旗杆，灯笼两个，也自取其乐。满城灯火不啻琉璃世界。
